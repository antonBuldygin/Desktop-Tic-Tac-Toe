Description

This is the final stage of our project. You need to add a menu for a quick game launch with a specific player configuration. Your program also lacks more detailed messages in the status bar about the current status of the game.

Objectives

Make the status bar more informative. We already have the Game in progress status; add a new message that displays the current turn: The turn of {0} Player ({1}). After the end of the game, the program should output the following message: The {0} Player ({1}) wins. You can use the MessageFormat class for this (take a look at the official Oracle doc for help). Instead of {0}, insert either Human or Robot. Instead of {1}, output either X or O.

Add a Game menu that includes five items:

  1. Human vs Human
  2. Human vs Robot
  3. Robot vs Human
  4. Robot vs Robot
  5. Exit

You can add a dividing line before the Exit item. You can use the following piece of code for this:

menuGame.addSeparator();
menuGame.add(menuItemExit);

When you select one of the first four menu items, the program must clear the field, set the desired player values in the toolbar, and start the game. When the game is on, the Start/Reset button must be set to Reset mode, the player buttons must be disabled, and the field cells must be enabled.

For testing purposes, the menu components should be named as follows:

  • MenuGame
  • MenuHumanHuman
  • MenuHumanRobot
  • MenuRobotHuman
  • MenuRobotRobot
  • MenuExit
